### 简化对象写法
```
在大括号里面，直接写入变量和函数，作为对象的属性和方法。

注意：对象简写形式简化了代码，所以以后用简写就对了
```

#### 代码
```js
    let name = 'zhangsan'
    let age = 18
    let func = function() {
        console.log(123)
    }
    let Obj = {
        name:name, //完整写法
        age, //简写
        func,
        change(){
            console.log('change')
        }
    }
    console.log(name) // zhangsan
    console.log(age) // 18
    Obj.func() // 123
    Obj.change() // change
```
