### 常规布局结构
```html
  <div class="container">
    <!-- 头部 -->
    <header class="header">
        <!-- logo图片 -->
        <span class="logo">
            <img src="" />
        </span>

        <!-- 导航 -->
        <nav class="nav">
            <a href="#">首页</a>
            <a href="#">关于我们</a>
            <a href="#">产品中心</a>
        </nav>
    </header>

    <!-- 中间内容 -->
    <section class="content">
        <div class="left">左边容器</div>
        <div class="right">右边容器</div>
    </section>

    <!-- 底部 -->
    <footer class="footer">
        <span>友情链接</span>
        <span>友情链接</span>
        <span>友情链接</span>
    </footer>
  </div>
```

### base标签
```html
  <!-- 为页面上的所有的相对链接规定默认目录位置 -->
  <head>
    <base href="/Volumes/dancework/html/images/">
  </head>
  <body>
    <img src="cover.jpg" />
  </body>
```
