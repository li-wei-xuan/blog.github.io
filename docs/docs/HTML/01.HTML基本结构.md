### HTML
```
HTML 是用来描述网页的一种语言
  HyperText Markup Language (超文本标记语言) 简称HTML
  HyperText: 超文本 (文本 + 图片 + 视频 + 视频 + 链接)
  Markup Language: 标记语言
```

### HTML 标签
```
HTML 标记标签通常被称为 HTML 标签 (HTML tag)。
  HTML 标签是由尖括号包围的关键词，比如 <html>
  HTML 标签通常是成对出现的，比如 <b> 和 </b>
  标签对中的第一个标签是开始标签，第二个标签是结束标签
  开始和结束标签也被称为开放标签和闭合标签

  标签名不区分大小写，但是XHTML中标签名必须用小写
```

### HTML 声明标签
```
<!DOCTYPE html> 声明是文档中的第一个成分，位于 <html> 标签之前
该标签告知浏览器文档所使用的 HTML 规范
```

### HTML 基本结构
```html
<!DOCTYPE html>
<!-- <html lang="en"> -->

<head>
    <!-- <meta charset="UTF-8">
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    <meta name="viewport" content="width=device-width, initial-scale=1.0"> -->
    <title>我的第一个网页</title>
</head>

<body>
    hello world!
</body>

</html>
```

### head 设置页面头消息
```html
设置页面标题：
  <title>我的第一个网页</title>

设置页面编码：
<meta http-equiv="Content-Type" content="text/html; charset=utf-8" />
  utf-8 多国语言编码
  gb2312 中文简体编码

设置页面关键字,内容介绍
      <meta name="keywords" content="关键字1,关键字2,..." />
      <meta name="description" content="这是一段描述信息..." />
      <meta name="author" content="XXXXX@qq.com" />
      <meta name="copyright" content="本网站版权归XXXXX所有" />
```
