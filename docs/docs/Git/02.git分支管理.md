### git branch
```
//列出分支
git branch
//创建一个分支
git branch (branchname)
//删除分支命令：
git branch -d (branchname)
```
### git checkout
```
//切换分支
git checkout (branch)
//创建新分支并切换到该分支
git checkout -b (branchname)
```
### git merge
```
//分支合并
git merge
```
### git rebase
```
//分支合并
git rebase
```
