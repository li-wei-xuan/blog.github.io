### vue中的循环

```js
//循环数组
  v-for="(value, index) in array"
  v-for="(value, index) of array"
// 循环对象 
  v-for="(value, index) in object"
  v-for="(value, index) of object"
// 循环字符串
  v-for="(value, index) in str"
  v-for="(value, index) of str"
  
循环数组、对象、字符串使用v-for...in和v-for...of，结果都是一致的
```

### for in 和 for of 的区别

```js
for...in 循环：只能获得对象的键名，不能获得键值
for...of 循环：允许遍历获得键值

遍历数组
	const person = [
    { id: '001', name: '张三', age: 18 },
    { id: '002', name: '里斯', age: 19 },
    { id: '003', name: '王武', age: 20 }
  ]
  
  for...in ：0 1 2
	for...of ：
		{ id: '001', name: '张三', age: 18 }
    { id: '002', name: '里斯', age: 19 }
    { id: '003', name: '王武', age: 20 }
    
遍历对象
	const car = {
    name: '奥迪A8',
    price: '70万',
    color: '黑色'
  }
  for...in ：
  	name
		price
		color
	for...of ：
		报错
		
遍历字符串
	const str='hello world'
	for...in ：0 1 2 3 4
	for...of ：h e l l o
```
