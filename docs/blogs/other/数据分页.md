## 分页
前端分页，后端分页

### 分页原理
```
分页显示就是将数据库中的数据集，分成一段一段的来显示.
这里需要两个初始的参数：
  每页的条数 $PageSize
  当前页面 $Page
```

### php实现分页
```php
// 主要是使用SQL的limit

// 列表
limit(($Page-1) * $PageSize,$PageSize)
// 总条数
count($list)
//总页数
ceil(count($list) / $PageSize)
```

### 前端实现分页
```js
// 主要是使用数组的slice

// 列表
data.slice((Page - 1) * PageSize, Page * PageSize)
// 总条数
data.length()
//总页数
ceil(data.length() / $PageSize)
```
